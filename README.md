Descrição do Projeto
O desenvolvimento deste projeto teve como objetivo aplicar na prática os conceitos aprendidos no Módulo 01 da Trilha HTML da DIO.me. A proposta foi criar uma página simples, mas completa, utilizando todas as tags apresentadas nas aulas, além de pesquisar e experimentar novas tags sugeridas.

Durante a construção da página, foram utilizados diferentes níveis de títulos (<h1> até <h6>) para organizar o conteúdo em seções hierárquicas. Os parágrafos (<p>) receberam destaques com <mark>, notas em <small>, além de estilos como <i>, <u> e <strong> para dar ênfase às informações. Também foram aplicados exemplos de fórmulas com <sub> e <sup>.

As listas ordenadas (<ol>) e não ordenadas (<ul>) ajudaram a estruturar tópicos e curiosidades de forma clara. Links (<a>) foram adicionados para direcionar o usuário a conteúdos externos, e citações (<blockquote>) enriqueceram o texto com frases inspiradoras. A tag <hr> foi usada para separar seções e melhorar a leitura.

Além disso, foram exploradas tags adicionais como <font> (mesmo obsoleta, para fins de aprendizado), <del> para indicar exclusões e <abbr> para explicar siglas. O tema escolhido para o conteúdo foi “Curiosidades sobre Inteligência Artificial”, trazendo informações reais da internet sobre a origem do termo, aplicações atuais e perspectivas futuras.

Esse processo permitiu consolidar o aprendizado das tags básicas de HTML e compreender como elas podem ser combinadas para criar páginas informativas e bem estruturadas. O desafio também incentivou a pesquisa e a prática, preparando o terreno para os próximos módulos, onde conceitos de CSS e JavaScript serão integrados para dar estilo e interatividade ao site.
